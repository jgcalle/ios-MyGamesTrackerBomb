iOS App My Video Game Tracker
=============================================
iOS 7 - xcode 5.0
 
Descripción

My Video Game Tracker permite llevar una gestión ágil y rápida de tu colección de videojuegos. Realiza una búsqueda para encontrar videojuegos, consulta la información detallada de los juegos encontrados (plataformas disponible, fecha lanzamiento, carátula, desarrollador…) y decide cual quieres agregar a tu colección.

Los resultados de las búsquedas se obtienen gracias a GiantBomb.com, la base de datos online más grande del mundo Gamer. De esta forma, sea cual sea el juego que busques te lo encontramos para que puedas agregarlo o bien a tu colección o a tu lista de favoritos (Must-Haves).
 
Desarrollo

El desarrollo se realizará con Xcode 5.0, y como opciones de despliegue de la aplicación se indica, aplicación para dispositivos iPhone con iOS 7 o superior y se permite cambio de orientación “portrait” y “landscape”.

A lo largo del desarrollo se ha prestado atención a la calidad del código, siguiendo la convención de código para Objective-C, pragma-mark, comentarios, convención de nombres en inglés, etc; y teniendo especial cuidado en seguir el patrón de arquitectura de software Modelo Vista Controlador (MVC), y otros patrones creacionales como Singleton y Factory. También se ha hecho uso de Bloques en llamadas asíncronas y uso de Categorías para ampliar el comportamiento de clases.
